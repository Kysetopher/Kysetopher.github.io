HeyJinn.ai

Hypothesis

A system designed for capturing users' thoughts and organizing them automatically is capable of assisting the user effectively. Advancements in artificial intelligence technology have made utilizing automated processes much more accessible. However, anticipating the users' needs is an essential part of being helpful and avoiding flooding the user with too many options. To collect the most important data, the system must be designed to collect input from the user easily.

Ease of use across devices
Recording thoughts and editing notes
Anticipation of user needs
Converstational data retrevial
Task automation and tracking

Design

Home

HeyJinn.ai has few components. The home screen contains a queue of potential items of interest: Suggested articles, reminders and important goals for the day.

Input Bar

The input bar remains at the bottom of the page allowing the user to send requests to thier agent at all times. As the user types into the input bar the home screen is filtered to show related suggested content and past records.

Navigation

Each item that is displayed can be opened to reveal its contents.

Editor

Data collected from users input is organized to build a user profile and support conversational retrieval.
